## Почему не VPN

<Warning>
**VPN запрещён для работы с аккаунтами.**
</Warning>

VPN-сервисы используют IP-адреса, которые Telegram уже знает:

- **Сгоревшие IP** — популярные VPN (NordVPN, ExpressVPN, Surfshark) используют одни и те же IP для миллионов пользователей. Telegram давно добавил их в свои базы.
- **Публичные списки** — IP-адреса VPN публикуются в открытых базах. Telegram проверяет их.
- **Требование Premium** — при входе с плохого IP Telegram может потребовать покупку Premium. Это прямой сигнал, что IP не подходит.

Используйте качественные прокси вместо VPN.

## Что такое прокси

Прокси — промежуточный сервер между вами и Telegram. Весь трафик идёт через него, и Telegram видит IP прокси, а не ваш.

В отличие от VPN, прокси работает только для конкретного приложения. Остальной трафик устройства идёт напрямую.

## Типы прокси

### SOCKS5

Универсальный протокол. Поддерживает аутентификацию (логин/пароль).

| Параметр | Значение |
|----------|----------|
| Поддержка Telegram | Полная |
| Шифрование | Нет (только аутентификация) |
| Скорость | Высокая |

### MTProto

Специальный протокол для Telegram. Маскирует трафик под обычный HTTPS.

| Параметр | Значение |
|----------|----------|
| Поддержка Telegram | Полная |
| Шифрование | Да |
| Скорость | Высокая |

<Note>
Некоторые MTProto-прокси добавляют рекламные каналы. Это не опасно, но раздражает. Выбирайте провайдера без рекламы.
</Note>

## Какой прокси выбрать

### По типу IP

| Тип | Траст | Цена | Когда использовать |
|-----|-------|------|-------------------|
| Резидентный | Высокий | $$$ | Важные аккаунты, долгосрочная работа |
| Мобильный | Средний-высокий | $$ | Хороший баланс цены и качества |
| Датацентровый | Низкий | $ | Только если нет других вариантов |

### Критерии выбора

- **Регион** — должен совпадать с регионом номера аккаунта.
- **Стабильность** — прокси не должен падать и менять IP.
- **Репутация** — IP не в чёрных списках.
- **Скорость** — задержки влияют на работу Telegram.

## Настройка в Telegram

1. Откройте **Настройки**.
2. Перейдите в **Данные и память** → **Тип соединения**.
3. Выберите **Использовать прокси**.
4. Нажмите **Добавить прокси**.
5. Выберите тип (SOCKS5 или MTProto).
6. Введите данные: сервер, порт, логин/пароль (для SOCKS5) или secret (для MTProto).

После настройки Telegram покажет статус подключения. Если прокси работает, появится галочка.

## Частые проблемы

### Прокси не подключается

- Проверьте данные (сервер, порт, логин, пароль).
- Убедитесь, что прокси работает (спросите у провайдера).
- Попробуйте другой прокси.

### Telegram требует Premium

IP прокси плохой (датацентр, в чёрных списках). Нужен другой прокси с лучшим IP.

### Частые запросы верификации

IP прокси вызывает подозрения. Смените на более качественный.

## Где брать прокси

Мы не рекламируем конкретных провайдеров. При выборе обращайте внимание на:

- Отзывы и репутацию.
- Наличие нужных регионов.
- Тип IP (резидентный, мобильный, датацентр).
- Возможность тестирования перед покупкой.

<Warning>
Не используйте бесплатные публичные прокси. Они нестабильны, медленны и часто собирают данные.
</Warning>
